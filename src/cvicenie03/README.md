# 3. Zadanie

## Časti zadania:
1. Načítajte reťazec R. Skopírujte ho do reťazca S tak, že všetky číslice obalíte do zátvorky. Reťazec S vypíšte.
2. Na planéte Nebucron sa za správnu vetu považuje taká, kde počet písmen A nie je väčší ako počet písmen O. 
   Napríklad PROGRAMMING IS NOT FOR YOU. je správna veta, ale SAY IT AGAIN! nie je. Načítajte vetu tvorenú písmenami veľkej abecedy a interpunkčnými znamienkami. 
   Zistite, či je v tomto zmysle správna. O svojom zistení informujte užívateľa!
