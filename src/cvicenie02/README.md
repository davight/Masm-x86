# 2. Zadanie

## Časti zadania:
1. Načítajte reťazec a vypíšte ho na obrazovku tak, aby každé písmeno reťazca bolo na samostatnom riadku.
2. Vypíšte reťazec na obrazovku tak, aby každé písmeno reťazca spolu s nasledujúcim písmenom v abecede bolo na samostatnom riadku. Napr. reťazec AHOJ sa vypíše takto: </br>
   AB </br>
   HI </br>
   OP </br>
   JK
3. Načítajte znak. Vypíšte počet výskytov zadaného znaku v reťazci.
